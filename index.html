Kuidas kasutada Google Classroomi

Järgi alltoodud samme — see võtab umbes 5 minutit.

M

Kasutad mobiili?

Laadi äpp alla enne alustamist — see on kiirem ja annab teavitusi.

  1. Laadi alla Google Classroom (App Store / Google Play)
  2. Logi sisse sama Google'i kontoga
  3. Kursus ilmub automaatselt pärast liitumist
Brauseris: kõikides Google'i teenustes (Gmail, Drive jne) on paremas ülanurgas ruudustiku ikoon — sealt leiad Classroomi.

Ametlik juhend: Google Classroomi äpp — Google tugi

1

Loo Google'i konto

Kui sul on juba gmail.com aadress — jäta see samm vahele ja mine sammu 2 juurde.
  1. Mine aadressile accounts.google.com/signup
  2. Sisesta oma nimi
  3. Vali kasutajanimi (nt eesnimi.perekonnanimi@gmail.com)
  4. Vali parool ja kinnita see
  5. Kinnita telefoninumbriga

Ametlik juhend: Google konto loomine — Google tugi

2

Liitu kursusega

  1. Mine classroom.google.com
  2. Klõpsa Logi sisse Google'iga
  3. Klõpsa paremas ülanurgas + nupul
  4. Vali Join class
  5. Sisesta liitumiskood, mille saatsime e-kirjaga:
Kood on saadetud e-kirjaga
Koodi ei leia? Vaata rämpsposti kausta. Ikka ei leia — kirjuta kaja.loorens@hkhk.ee

Ametlik juhend: Kursusega liitumine — Google tugi

3

Leia materjalid

  1. Ava kursus klõpsates selle pealkirjal
  2. Klõpsa üleval Classwork
  3. Materjalid on jagatud teemade kaupa — klõpsa teema pealkirjal et avada
  4. Slaidid ja juhendid avanevad uues aknas
Materjalid ilmuvad jooksvalt — iga sessiooni eel lisame uue teema.

Ametlik juhend: Materjalide vaatamine — Google tugi

4

Esita ülesanne ja küsi küsimusi

  1. Klõpsa Classwork → leia ülesanne
  2. Ava ülesanne klõpsates pealkirjal
  3. Loe juhend läbi
  4. Lisa oma töö (vt allpool)
  5. Klõpsa Turn in ja kinnita
Screenshot või pilt

Klõpsa Add or create → File → laadi pilt üles → Turn in

Fail (PDF, dokument, konfifail)

Klõpsa Add or create → File → laadi fail üles → Turn in

Link

Klõpsa Add or create → Link → kleebi link → Turn in

Lugemis- või vaatamisülesanne

Klõpsa Mark as done kui oled materjali läbinud

Ülesannetel võib olla näidatud 100 punkti — ära muretse, seda koolitust ei hinnata punktidega. Oluline on et töö on esitatud.

Küsimused ülesande kohta

Privaatne küsimus — ainult koolitaja näeb

Ülesande all → Add private comment

Avalik küsimus — kõik osalejad näevad

Ülesande all → Class comments väli

Teadaanne kõigile — Stream

Stream sakk → Share something with your class

Ametlik juhend: Ülesande esitamine — Google tugi · Kommentaarid — Google tugi

5

Tähtajad ja teavitused

Tähtajad ilmuvad automaatselt Google Calendaris

Kui ülesandel on tähtaeg, näed seda kohe oma kalendris — ei pea midagi lisama.

Lülita sisse teavitused

Classroomis paremas ülanurgas kella ikoon → vali mis teavitusi soovid saada.

Kõik esitamata ülesanded ühes kohas

Classroomi avaleht → To-do — näed kõiki tähtaegu kõigilt kursustelt.

Ametlik juhend: Kalendri kasutamine — Google tugi · Teavitused — Google tugi
